Kindlefs

FuseFS to manage Kindle Collection Data

Download this project as a .zip file Download this project as a tar.gz file

KindleFS is a FuseFS that mounts a Kindles data via Rindle and allows the management of the kindles collection information via filesystem tools.

Installation

Add this line to your application's Gemfile:

gem 'kindlefs'

And then execute:

$ bundle

Or install it yourself as:

$ gem install kindlefs

Usage

Just mount the kindle into some folder.

$ kindlefs /path/to/kindle/root /path/to/mountpoint

After you are done, unmount the kindlefs via CTRL-C and unmount the kindle. Then do a hard-reset, holding the power-slider for about 20 seconds.
